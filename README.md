# xcode-ios7-colors

Xcode syntax highlighting iOS7 color scheme

## Quick Install

    curl https://raw.github.com/dpree/xcode-ios7-colors/master/iOS%207.dvtcolortheme > "$HOME/Library/Developer/Xcode/UserData/FontAndColorThemes/iOS 8.dvtcolortheme"

## Local Development

### Install

    ./install.sh

### Backup

    ./install.sh backup
